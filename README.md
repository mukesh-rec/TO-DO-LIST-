# TO-DO-LIST-
a console based TO DO LIST program using core java and array list.
*TECHNOLOGIES USED* - Core java, Object - Oriented Programming (OOP), ArrayList, File I/O.
*HOW TO RUN* - the scanner display will show all the actions and will guide you to enter the numbers which change the actions, based on the input the user can see the corresponding output.
*KEY FEATURES* - the user can add, view, mark complete, remove the tasks and saving the tasks using array list
